2015.5.25 开放学院的第一个互联网开源项目beta版发布。


version:   
记录并探讨以建筑作品为索引的线上社区如何改变着传统建筑设计的行为。

scope:  
涉及O2O社区、开放学院、云工作室，每一部分都与互联网或移动互联网有深度的融合，"互联网+"正在静悄悄的改变着由柯布西埃、格罗皮乌斯、密斯、赖特等现代主义先锋发展起来的建筑设计行为。





**云工作室**  
ikuku云工作室基于大数据及云办公的优势，在合理的成本范围内，协助建筑师提高项目的完成度。
